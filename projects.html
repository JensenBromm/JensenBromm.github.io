Java Projects

My favorite Java project is my Flappy Bird clone

For this game, I used Java3D to model the pipes, the code for the pipes can be viewed here.
To simplify the code, I decided to only move the player up and down. The pipes are moving horizontally across the screen which gives the illusion of player movement.
Once the pipes are off-screen, they are then deleted from the game and recreated at a different position so that the game can loop indefinitely.
